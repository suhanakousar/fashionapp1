## üìπ Recording Setup

### Screen Recording Software
- **Windows**: OBS Studio, Windows Game Bar (Win+G)
- **Mac**: QuickTime, OBS Studio, ScreenFlow
- **Online**: Loom, Screencastify

### Recommended Settings
- **Resolution**: 1920x1080 (Full HD)
- **Frame Rate**: 30fps (60fps optional)
- **Audio**: 44.1kHz, 128kbps
- **Format**: MP4 (H.264)

### Browser Setup
- Use Chrome or Firefox (best compatibility)
- Zoom level: 100% (not zoomed in/out)
- Full screen or large window
- Clear browser cache before recording

## üé® Visual Tips

### Highlighting
- Use cursor to point at important elements
- Slow down when showing key features
- Pause briefly on explainability panel
- Smooth mouse movements (no jittery cursor)

### Transitions
- Smooth page navigation
- No abrupt cuts
- Natural flow between features
- Show loading states (adds realism)

### Focus Areas
- **0:20**: Kiroween theme (dark palette, neon accents)
- **1:45**: Explainability panel (judge-friendly transparency)
- **2:45**: `.kiro` folder (demonstrates Kiro usage)

## üö® Common Mistakes to Avoid

1. **Too Fast** - Slow down, let judges see features
2. **No Narration** - Always explain what you're doing
3. **Poor Audio** - Test microphone, use quiet space
4. **Low Resolution** - Record in at least 1080p
5. **Missing Features** - Show all key features
6. **No Kiro Highlight** - Always show `.kiro` folder
7. **Rushed Ending** - End with clear call-to-action
